BookReview APIs

GET /api/reviews

Get review list.

POST /api/reviews

Create a new review.

Required Parameters:

  • title: (string) Book title
  • body: (string) Review comment
  • score: (number) Review score
  • reviewer: (string) Reviewer name

PUT /api/reviews/:id

Update a review.

Required Parameters: Same with Create ones.

DELETE /api/reviews/:id

Delete a review.
